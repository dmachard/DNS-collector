# DNS-collector - Supported collectors

| Loggers                                    | Descriptions                                          |
| :------------------------------------------|:------------------------------------------------------|
| [DNStap](collectors/collector_dnstap.md)              | DNStap receiver and proxifier |

| [File Ingestor](collectors/collector_fileingestor.md)         | File ingestor like pcap |
